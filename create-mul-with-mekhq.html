Some Assumptions

This guide assumes you know the basic features of MHQ & how to add and customize MechWarriors and ‘techs. If you have any questions, please do ask us on the Discord!

Step 1 – MHQ Initial Setup

Create your MekHQ save file. Set the date to 01/01/3062. Each screenshot below shows the settings used for each tab.

N.B: The finances tab is setup this way as I will do all additions/subtractions of funds using GM mode on the finances tab in MHQ. This will come later in the guide.

You can ignore the Mercenary, Experience, Special Abilities, Personnel Market, Rank System, Against the Bot, and Skill Randomization tabs.

Set name and portrait generation to use whatever settings you prefer.

Step 2 – GM Mode, Finances, and the Stable Creation Rules

Ensure you have GM mode enabled in the top right.

Assign your starting budget in the Finances tab using the ‘Add Funds (GM)’ button:

Use the ‘Starting Capital’ type so you can see the source of funds easily in the ledger.

Follow the stable creation rules for your stable as per the Solaris rules document. Use the Excel cost calculator to work out the total amount to be subtracted.

Subtract your costs from the starting budget using the Add Funds (GM) button by inputting negative numbers:

This is the same process you will use after battles and at the end of each week to subtract your repair and administration costs, as well as to add any winnings.

Step 3 – Creating a ‘Mission’ and Assigning Forces

On the ‘Briefing’ tab, hit ‘Add Mission’ and enter the below in the dialogue box:

Set the date in MekHQ to the date you want the scenario to take place on by using the ‘Advance Day’ button. Due to how we are managing the tournament, set the day of your seed match to the day you will play i.e. Saturday/Sunday.

N.B – once ladder matches begin, your GM will let you know when to synchronize dates in MekHQ across all save files.

Hit the ‘Add Scenario’ button in the briefing room, making sure you have the right weight class selected:

Assign a unit from your TO&E as appropriate:

The assigned unit appears in the briefing room – click ‘Export MUL File’:

Save the MUL file somewhere you can easily find it. In MegaMek, click the ‘Load Unit List’ button and select your MUL file:

Your units will appear in MM:

Select a map, and play the game as normal. When the game ends, click ‘Yes’ on the option to save a salvage file:

I like to save the file with the same name as the mission with ‘- OUTCOME’ added to the end of the file name:

In MekHQ, hit ‘Resolve Manually’:

Go through the steps for resolving.

Importantly:

  • DO NOT say ‘YES’ to controlling the battlefield – no salvage here!
  • DO un-tick ‘TOTAL LOSS’ next to any unit destroyed, so that you can keep it and can roll the repairs using the Solaris rules.
  • DO un-tick any pilots as KIA so you can manage death/injury using the Solaris rules.

Then follow the Solaris rules for all post battle results, including earnings, expenditure and add/remove funds as appropriate on the finances screen.

Follow the same method above for all ladder, exhibition, team, and grudge matches played. Do not include sim/training/testing battles.
